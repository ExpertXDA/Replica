# Replica

Локальный AI-компаньон для ПК с полноценным GUI и фоновым режимом.

## Что реализовано

- Полноценный запуск как GUI-приложения (`Tkinter`), без консольного интерфейса ввода.
- Автоматический запуск локальной LLM-модели: при старте проверяется Ollama, при необходимости поднимается `ollama serve` и подтягивается выбранная модель.
- Постоянный overlay-ассистент поверх окон: аватар, эмоция, текст, drag-and-drop перемещение, прозрачность.
- Система аватаров через `assets/avatars/`:
  - если есть `neutral.png`, `happy.png`, ... — используются изображения;
  - если нет — автоматически fallback на emoji.
- Непрерывное голосовое распознавание с wake-word (`Replica`) и настраиваемой чувствительностью.
- Ответ ассистента мужским голосом (приоритет выбора мужского TTS-голоса) с настройкой скорости речи.
- Фоновая работа анализатора экрана и контекстные реакции.
- Голосовые/текстовые системные команды: открыть браузер, запуск приложения, скриншот, power-сценарии с защитой.
- Расширенные настройки в GUI-панели: AI-модель, режим производительности, voice rate, чувствительность, overlay-параметры.

## Быстрый старт

```bash
python Main.py
```

## Структура

```text
assets/avatars/
core/{brain,memory,speech,vision}
system/{commands,config,performance}
ui/{avatar,emotions,window}
```

## Примечания

- Для лучшего качества локального LLM-ответа нужен установленный Ollama.
- Для голосового ввода нужен рабочий микрофон и `speech_recognition`.
