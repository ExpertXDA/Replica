# Replica

Локальный AI-компаньон для ПК с модульной архитектурой и оффлайн-first подходом.

## Реализовано

- Модульная структура проекта (`core`, `ui`, `system`, `plugins`, `assets`).
- Конфигурация режимов производительности (`eco`, `balanced`, `max`).
- Загрузка/сохранение настроек из `config/settings.json`.
- Базовая память (краткосрочная и долгосрочная).
- Personality prompt + `AssistantBrain`.
- LLM-адаптер с local-first стратегией (Ollama -> fallback).
- STT/TTS адаптеры (локальные fallback'и).
- Захват и анализ экрана (`core/vision/screen_analyzer.py`) с настраиваемым интервалом.
- Always-on-top overlay окно (`Tkinter`).
- UI-панель настроек (`/settings`).
- Роутер системных голосовых/текстовых команд (браузер, скрин, power, volume).

## Запуск

```bash
python Main.py
```

Команды в рантайме:

- `/settings` — открыть панель настроек.
- `/exit` — завершить работу.

## Примечания

- Для локальной генерации через LLM требуется запущенный Ollama на `127.0.0.1:11434`.
- При отсутствии внешних зависимостей система продолжает работать в fallback-режиме.
