# Replica

Локальный AI-компаньон для ПК с модульной архитектурой.

## Что уже есть

- Базовая модульная структура (`core`, `ui`, `system`, `plugins`, `assets`).
- Конфигурация режимов производительности: `eco`, `balanced`, `max`.
- Система эмоций и emoji-аватар как fallback.
- Краткосрочная и долгосрочная память.
- Базовый "мозг" ассистента с зачатками поведенческой модели.
- Демо-режим в терминале для локальной проверки логики диалога.

## Структура

```text
Replica/
  core/
    brain/
    speech/
    vision/
    memory/
  ui/
    avatar/
    window/
    emotions/
  system/
    performance/
    config/
  plugins/
  assets/
  Main.py
```

## Запуск

```bash
python Main.py
```
